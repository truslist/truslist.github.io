TrusList

An Open and Trustworthy Domain Ranking System

This is the official project page for **TrusList**, our novel multi-dimensional evaluation framework for domain top lists. TrusList addresses the challenges of manipulation susceptibility, opaque ranking criteria, and geographic biases found in conventional domain ranking systems.

Key Features & Contributions:

✓ Multi-dimensional Evaluation: Integrates spatio-temporal query characteristics, inter-domain dependencies, and registrar reputation.
✓ Enhanced Reliability: Achieved through cross-validation and integration with large-scale passive DNS data.
✓ Broader Coverage & Stability: Demonstrates superior performance in retaining regionally significant domains.
